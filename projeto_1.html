Explicando as tags apresentadas nas aulas.

Começando pela tags de cabeçalhos (headings), que vai de <H1> a <H6>, seguindo do mais importante ao menos importante em ordem decrescente. Utilizados para definição do titulo e subtitulos da estrutura do site.

A tag <p>, é usada para a criação dos paragrafos.

Seguido pela tag <hr>, que cria uma linha horizontal como mostra logo abraixo.

Ênfase e Formatação:

A tag <em> e a tag <i> possuem um resultado visual ideintico, porem o significado semântico é diferente. A tag <em> representa ênfase importante de seus conteúdos, enquanto que a tag <i> representa o texto que é iniciado de uma prosa, como uma palavra estrangeira, pensamentos de um personagem ficcional, ou quando o texto se refere á definição de uma palavra em vez de representar seu significado semântico.

A tag <strong> é usada para exibir o texto em negrito, dando ênfase ao texto em partes importantes, ao contrario da tag <b> que apenas deixa o texto visualmente em negrito, mas ainda usado em partes importantes que precisam ser destacadas.

A tag <mark> como o próprio nome sugere, faz com que o texto fique marcado.

A tag <small>, deixa o texto menor, geralmente usado em notas nos rodapes das paginas, avisos legais, etc.

A tag <u>, deixa o texto sublinhado como nesse exemplo.

Tags usadas para criar listas

Existem dois tipos de listas, ordenadas <ol> que pode ser exibida com numeros arábicos, letras, e numeros romanos. E as listas não ordenadas <ul>, que são exibida com marcadores.

Para selecionar o tipo de indicativo que tera a lista ordenada devemos fazer o seguinte comando: <ol type="">, dentro das aspas dupas colocaremos (1,a,A,i ou I) para selecionar o tipo.

  1. item 1
  2. item 2
  3. item 3
  1. item 1
  2. item 2
  3. item 3

A seguir dentro da lista usamos a tag <li> para criar os itens da lista, alem de ser possivel usar tag <ol> e <ul> dentro dela mesma para criar sub itens da lista.

  1. item 1
    1. item 1.1
    2. item 1.2
    3. item 1.3
  2. item 2
  3. item 3

Links e citações:

A tag <blockquote> serve para criar citações em blocos, usadas para criar citações mais longas como no exemplo abaixo.

"Lorem ipsum dolor sit, amet consectetur adipisicing elit. Mollitia, iure laborum. Ratione dolore sint fuga possimus minima ab a. Accusamus aut eius dolores laudantium sit mollitia neque voluptas voluptatum nemo. Lorem ipsum, dolor sit amet consectetur adipisicing elit. Harum eum eaque similique ipsam repudiandae enim modi aliquid. Provident tenetur laudantium, autem iure eum maxime porro odit sunt sapiente perspiciatis ad".

A tag <a> (anchor), é usada para ancorar links para outras paginas ou sites, até mesmo para um lugar especifico do texto na mesma pagina.

Subscrito e Sobrescrito:

A tag <sub> (subscript), é bastante utilizada em formulas de química como H2O, ou em notas de rodapé.

A tag <sup> (superscript), é geralmente usada para expoentes (X2).

Novas tags sugeridas no exercício:

A tag <font>, atualmente está obsoleta, era usada para definir o estilo da fonte (tipo, tamanho, cor) em HTML. No entanto, ela foi declarada obsoleta e não é mais recomendada para uso, assim utilizando o CSS para fazer as estilizações necessarias.

A tag <del> indica que uma parte do texto foi excluída ou removida de um documento. O texto dentro da tag geralmente é exibido com um risco horizontal no meio. como no exemplo a seguir.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Impedit, delectus aperiam sit quod quos officiis corrupti necessitatibus modi nulla fuga, doloribus quam, iste blanditiis quia maiores dolorum. Perspiciatis, sit quaerat.

A tag <abbr> cria uma abreviação, para uma frase ou palavra, uma muito utilizada atualmente é OBG, e ao passar o ponteiro por cima dela é possíel ver seu significado completo.
